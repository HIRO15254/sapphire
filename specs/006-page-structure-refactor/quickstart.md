# クイックスタート: ページ構造リファクタリング

**Feature**: 006-page-structure-refactor
**Date**: 2025-12-05

## 概要

このアップデートでは、アプリケーションのナビゲーション構造が改善されました。主な変更点：

- サイドバーナビゲーションの追加
- トップページがダッシュボードに変更
- セッションの詳細表示ページが追加

## 新しいナビゲーション

### サイドバー

ログイン後、画面左側にサイドバーが表示されます。サイドバーから以下のページに直接アクセスできます：

| アイコン | リンク | 説明 |
|---------|--------|------|
| 🏠 | ダッシュボード | 統計情報と最近のセッションを表示 |
| 📋 | セッション一覧 | すべてのセッションを表示・検索 |
| ➕ | 新規作成 | 新しいセッションを記録 |

### サイドバーの折りたたみ

- **デスクトップ**: サイドバー上部のボタンで展開/折りたたみを切り替え
- **モバイル**: ハンバーガーメニュー（☰）をタップしてサイドバーを開閉

## ダッシュボード

ログイン後、最初に表示されるダッシュボードでは以下の情報を確認できます：

### 統計情報
- 総利益/損失
- セッション数
- 勝率
- 平均利益/損失

### 最近のセッション
直近5件のセッションが表示されます。クリックすると詳細ページに移動します。

### クイックアクション
「新規セッション作成」ボタンから、すぐにセッション記録を開始できます。

## セッション操作

### セッションの詳細を見る

1. セッション一覧またはダッシュボードでセッションをクリック
2. 詳細ページで以下の情報を確認：
   - 日時、場所
   - バイイン、キャッシュアウト、収支
   - プレイ時間
   - タグ
   - メモ

### セッションを編集する

1. セッション詳細ページで「編集」ボタンをクリック
2. フォームで内容を修正
3. 「保存」ボタンで変更を反映

### セッションを削除する

1. セッション詳細ページで「削除」ボタンをクリック
2. 確認ダイアログで「削除」を選択
3. セッション一覧ページにリダイレクト

### 新しいセッションを記録する

1. サイドバーの「新規作成」またはダッシュボードの「新規セッション作成」をクリック
2. フォームに必要事項を入力：
   - 日時
   - 場所（既存から選択または新規作成）
   - バイイン額
   - キャッシュアウト額
   - プレイ時間
   - タグ（任意）
   - メモ（任意）
3. 「保存」ボタンでセッションを記録

## URL構造

各ページには固有のURLがあり、ブックマークや共有が可能です：

| URL | ページ |
|-----|--------|
| `/` | ダッシュボード |
| `/poker-sessions` | セッション一覧 |
| `/poker-sessions/new` | 新規作成 |
| `/poker-sessions/123` | セッションID 123の詳細 |
| `/poker-sessions/123/edit` | セッションID 123の編集 |

## トラブルシューティング

### サイドバーが表示されない

- ログインしているか確認してください
- ログインページ（/auth/signin）ではサイドバーは表示されません

### ページが見つからない（404エラー）

- セッションが削除されている可能性があります
- セッション一覧ページに戻って、該当のセッションが存在するか確認してください

### 編集・削除ができない

- 他のユーザーが作成したセッションは編集・削除できません
- 自分が作成したセッションのみ操作可能です
