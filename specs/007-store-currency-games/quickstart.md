# クイックスタートガイド: 通貨・ゲーム登録機能

**機能ブランチ**: `007-store-currency-games`
**作成日**: 2025-12-05
**対象ユーザー**: ポーカープレイヤー

## この機能でできること

日本のアミューズメントポーカーで使用される**通貨**（ポイント、チップなど）と**ゲーム**（NLHEリングゲーム）を登録し、セッション記録に紐付けることができます。

### 主な機能

- 🪙 **通貨の登録**: GGポイント、JOPTポイントなど、各種通貨を登録
- 🎮 **ゲームの登録**: 各店舗のゲーム（ステークス、ルール等）を登録
- 📊 **ゲーム別統計**: どのステークスで勝っているかを分析
- 💹 **通貨別統計**: 通貨ごとの収支を確認

---

## 1. 通貨を登録する

通貨は店舗とは独立して管理されます。同じ通貨を複数の店舗で使用できます。

### 操作手順

1. **設定画面**を開く
2. **通貨管理**を選択
3. 「**通貨を追加**」をクリック
4. **通貨名**を入力（例: 「GGポイント」「JOPTポイント」）
5. 「**保存**」をクリック

### ヒント

- 複数店舗で共通の通貨（JOPTポイント等）は一度登録すれば、すべての店舗で使用できます
- 通貨名はあとから編集可能です
- ゲームで使用中の通貨は削除できません

---

## 2. ゲームを登録する

ゲームは各店舗ごとに登録します。ゲームには使用する通貨を指定します。

### 操作手順

1. **店舗詳細画面**を開く
2. **ゲーム**タブを選択
3. 「**ゲームを追加**」をクリック
4. 以下の情報を入力:
   - **ゲーム名**: 例）「1/2 NL」「2/5 NL」
   - **通貨**: 登録済みの通貨から選択
   - **SB（スモールブラインド）**: 例）1
   - **BB（ビッグブラインド）**: 例）2
   - **Ante**: 任意（0 = なし）
   - **最小バイイン**: BB単位で入力（例: 40 = 40BB）
   - **最大バイイン**: BB単位で入力（例: 200 = 200BB）
   - **その他のルール**: 任意（ストラドルルール等）
5. 「**保存**」をクリック

### 入力例

| 項目 | 入力値 |
|------|--------|
| ゲーム名 | 1/2 NL |
| 通貨 | GGポイント |
| SB | 1 |
| BB | 2 |
| Ante | 0 |
| 最小バイイン | 40 |
| 最大バイイン | 200 |

### ヒント

- 先に通貨を登録しておく必要があります
- BBはSB以上の値を入力してください
- 最大バイインは最小バイイン以上の値を入力してください
- もう使わないゲームは「アーカイブ」すると、新規セッション作成時に表示されなくなります

---

## 3. セッションにゲームを紐付ける

セッション記録時に、どのゲームでプレイしたかを選択できます。

### 操作手順

1. **新規セッション作成**画面を開く
2. **店舗**を選択
3. **ゲーム**ドロップダウンが表示される
4. プレイしたゲームを選択（任意）
5. その他の情報を入力して保存

### ヒント

- ゲーム選択は任意です。選択しなくてもセッションは保存できます
- 店舗にゲームが登録されていない場合は、ゲーム選択欄は表示されません
- アーカイブされたゲームは選択候補に表示されません

---

## 4. ゲーム別・通貨別の統計を確認する

ゲームや通貨ごとの収支を確認できます。

### 操作手順

1. **統計画面**を開く
2. **ゲーム別**タブを選択
3. 各ゲームの収支、セッション数、勝率が表示される
4. 特定のゲームでフィルターすることも可能

### 表示される統計

| 項目 | 説明 |
|------|------|
| 収支 | そのゲームでの累計収支 |
| セッション数 | プレイしたセッション数 |
| 勝率 | 収支がプラスだったセッションの割合 |
| 平均収支 | 1セッションあたりの平均収支 |

## よくある質問

### Q: 通貨を削除したいのですが、削除ボタンが押せません

**A**: その通貨がゲームで使用されている場合は削除できません。先にその通貨を使用しているゲームをすべて削除するか、別の通貨に変更してください。

### Q: ゲームを削除したいのですが、削除ボタンが押せません

**A**: そのゲームがセッションで使用されている場合は削除できません。代わりに「アーカイブ」機能をご利用ください。アーカイブすると、新規セッション作成時には表示されなくなりますが、過去のセッションデータは保持されます。

### Q: 同じ通貨を複数の店舗で使いたいのですが

**A**: 通貨は店舗とは独立して管理されているため、一度登録した通貨は複数の店舗のゲームで使用できます。

### Q: アーカイブしたゲームを復活させたい

**A**: ゲーム一覧で「アーカイブ済みも表示」を有効にし、該当ゲームの「アーカイブ解除」を選択してください。

### Q: 既存のセッションにゲームを紐付けたい

**A**: セッションの編集画面から、ゲームを選択して保存してください。

## 設計のポイント

### なぜ通貨と店舗が分離されているのか

日本のアミューズメントポーカーでは、JOPT（Japan Open Poker Tour）ポイントのように、複数の店舗で共通して使用される通貨が存在します。通貨を店舗から独立させることで、このような複数店舗共通の通貨を効率的に管理できます。

### なぜゲームは削除ではなくアーカイブなのか

過去のセッションデータの整合性を保つため、セッションが紐付いているゲームは削除できません。代わりにアーカイブ機能を使うことで、新規セッション作成時には表示されなくなりますが、過去のデータは正確に保持されます。
