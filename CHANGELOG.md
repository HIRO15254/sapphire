# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.3.0] - 2026-02-01

### Added

- アカウント設定機能
  - アカウント設定ページの追加（連携済みログイン方法・パスワード管理）
  - OAuth アカウント連携 API の追加
  - アカウント連携解除・パスワード設定/変更サーバーアクションの追加
  - 連携済みアカウント照会用 tRPC ルーターの追加
  - サイドバーにアカウントナビゲーション項目を追加
  - Discord OAuth のメールアカウントリンクを有効化
- セッション詳細ページのモバイル対応リワーク
  - セッションヘッダー・サマリー・情報カード・オールインセクションのモバイル最適化
  - タイムライングルーピングロジックとイベントタイムラインセクションの追加
  - Server Actions ベースへの書き換え
- Linear 連携ワークフロー自動化システムの追加

### Changed

- 通貨管理機能
  - 通貨管理モジュールの抽出とドロワー式作成 UI への変更
  - 通貨のサーバーサイド一括取得・クライアントサイドフィルタリングに変更
  - 通貨 UI の英語化・カードレイアウトのコンパクト化

### Fixed

- オールインモーダルの「X 回実行」トグルが動作しない問題の修正
- 同卓者名の空欄処理を改善（プレースホルダー表示に変更）
- 自分着席（セルフシーティング）UI の追加と関連 API の修正
- ライブセッションボタンのタップターゲットサイズの改善
- サイドバー・ヘッダーレイアウトの改善（テーマ切替移動・タイトル折り返し防止）
- サイドバーナビゲーションの英語化
- セッションカードの右側要素の折り返し防止

## [1.2.0] - 2026-01-30

### Added

- トーナメントライブセッション機能
  - ブラインドタイマーカード（現在レベル・残り時間・次レベル表示）
  - トーナメントサマリー（参加人数・プライズプール・平均スタック・BB表示）
  - トーナメントスタックチャート
  - トーナメント情報タブ（詳細の閲覧・編集）
  - エントリー数・残り人数のインライン更新
  - 最終順位の記録
  - アドオン記録（コスト・獲得チップ）
  - 初期スタック入力（バイインとは別）
  - 既存トーナメント選択時のバイイン・初期スタック自動入力
- セッション一覧フィルター機能（ゲームタイプ・期間・通貨・店舗）
- 新規セッション作成ドロワー（ボトムシート UI）
- ヘッダーへの動的ページタイトル表示

### Changed

- セッション履歴カードを2行コンパクトレイアウトに刷新
- Cash/Tournament バッジを GameTypeBadge コンポーネントに統一
- セッション関連コンポーネントを features/sessions モジュールに整理
- React Cosmos 依存関係の削除

### Fixed

- アクティブセッションでチャートがはみ出す問題
- トーナメントのアドオンでコストではなくチップ数でスタック計算するよう修正
- プレイヤー人数入力で一時的に空欄にできない問題
- ポジション選択の順序と回転方向の修正（BTN始点・時計回り）
- NewSessionForm の useMemo 依存配列の修正

## [1.1.0] - 2026-01-12

### Added

- セッションライブ記録機能
  - ハンドカウンターに最終記録時間・ポジションの保存機能を追加

### Changed

- セッションライブ記録機能
  - グラフ・セッションサマリの表示を改善
  - タイムラインの表示・編集を改善
  - 同卓者の永続化の利便性向上
  - 同卓者に対するタグ割り当ての利便性向上
  - その他軽微なアイコン・カラーリングの改善
- セッション記録機能
  - セッション詳細画面のレイアウト・カラーリング改善

### Fixed

- アップデートが通知されない問題の解消
- セッションライブ記録機能
  - 非永続化プレイヤーの名前変更が反映されない問題の解消
  - 同一名称のユーザーが複数作成できる問題の解消
  - 卓リセット時に無数の通知が来る問題の解消
- その他開発上の不具合の解消
