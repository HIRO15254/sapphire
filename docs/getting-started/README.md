# Getting Started（スタートガイド）

新規参加メンバーがプロジェクトを理解し、開発環境をセットアップするためのドキュメント集です。

## 📋 読む順序

### 1. プロジェクト理解フェーズ
1. **[project-overview.md](project-overview.md)** - まずはここから
   - プロジェクトの目的と概要
   - 技術スタックの理解
   - 開発方針の確認

### 2. 環境構築フェーズ
2. **[database-setup-guide.md](database-setup-guide.md)** - 環境セットアップ
   - SQLiteデータベースの設定
   - 開発環境の構築
   - 動作確認手順

## ⏱️ 所要時間目安

- **プロジェクト理解**: 約30分
- **環境構築**: 約60分
- **動作確認**: 約15分

**合計**: 約1時間45分

## ✅ チェックリスト

環境構築完了の確認項目：

- [ ] プロジェクト概要を理解した
- [ ] 技術スタックを把握した
- [ ] 開発環境が正常に動作する
- [ ] データベースに接続できる
- [ ] サンプルデータの作成・表示ができる
- [ ] テストが実行できる

## 🚀 次のステップ

環境構築が完了したら、以下のドキュメントに進んでください：

1. **[../development/](../development/)** - 開発ガイド
2. **[../testing/](../testing/)** - テスト関連ドキュメント

## ❓ よくある質問

### Q: 環境構築でエラーが発生します
A: `database-setup-guide.md`のトラブルシューティングセクションを確認してください。

### Q: 動作確認で期待通りに動きません
A: 以下を確認してください：
- 依存関係のインストール状況
- データベースファイルの存在
- ポート番号の競合

### Q: 開発の進め方がわかりません
A: `../development/playernote-development-guide.md`を参照してください。
