# 💎 Sapphire

ライブポーカーにおける成績向上のためのツール群を提供するクロスプラットフォームデスクトップアプリケーションです。
Tauriフレームワークを使用し、React + TypeScript + Rust の組み合わせで、軽量かつ高速なネイティブアプリを実現しています。

## 🚀 特徴

- **軽量**: OSネイティブWebViewを使用し、Electronより高速・省メモリ
- **安全**: Rustの型安全性とTauriのサンドボックス化でセキュリティ強化
- **モダン**: React 19 + TypeScript 5.8 + Vite 7 の最新技術スタック
- **クロスプラットフォーム**: Windows・macOS・Linux対応

## 🎯 主要機能

### PlayerNote機能
ライブポーカーでの他プレイヤー情報管理システム
- **プレイヤー管理**: 対戦相手の基本情報とプレイスタイルを記録
- **ラベル機能**: プレイヤーを色付きラベルで分類（TAG、LAG等）
- **タグ機能**: 複数の特徴タグを組み合わせて詳細な分類が可能
- **ノート機能**: 自由形式でプレイヤーの行動パターンやメモを記録
- **検索・フィルタ**: 蓄積した情報から素早くプレイヤーを検索

## 📋 必要な環境

- **Node.js** 18+ 
- **Rust** 1.70+
- **Bun** (パッケージマネージャー)

## ⚡ クイックスタート

```bash
# 依存関係のインストール
ban install

# 開発サーバー起動
ban dev

# Tauriアプリケーション起動（別ターミナル）
ban tauri dev
```

## 📝 利用可能なコマンド

| コマンド              | 説明            |
|-------------------|---------------|
| `ban dev`         | Vite開発サーバー起動  |
| `ban build`       | プロダクションビルド    |
| `ban preview`     | ビルド結果をプレビュー   |
| `ban tauri dev`   | Tauriアプリ開発モード |
| `ban tauri build` | Tauriアプリをビルド  |
